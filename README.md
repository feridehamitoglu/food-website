# 🥗 Foody - Organik Gıda Restaurant Websitesi

Modern ve tam responsive bir organik gıda / restaurant tanıtım websitesi. HTML, CSS ve JavaScript kullanılarak sıfırdan geliştirilmiştir.

---

## 📸 Önizleme

> Projeyi klonladıktan sonra `index.html` dosyasını tarayıcıda açarak önizleyebilirsiniz.

---

## ✨ Özellikler

- 📱 **Tam Responsive Tasarım** — Mobil, tablet ve masaüstü uyumlu
- 🎯 **Sabit (Sticky) Header** — Kaydırma sırasında arka plan değişimi efekti
- 🍔 **Hamburger Menü** — Mobil cihazlar için açılır/kapanır navigasyon
- 🎬 **ScrollReveal Animasyonları** — Aşağıdan, soldan, zoom ve kademeli giriş efektleri
- 🛍️ **Ürünler Bölümü** — Kategori filtreleme butonları ve ürün kartları
- ⭐ **Müşteri Yorumları** — Hover efektli yorum kartları
- 📝 **Blog Bölümü** — Makale kartları
- 📬 **Bülten Aboneliği** — Footer'da e-posta kayıt formu
- ⬆️ **Yukarı Kaydır Butonu** — Animasyonlu ve otomatik görünen buton

---

## 🛠️ Kullanılan Teknolojiler

| Teknoloji | Açıklama |
|---|---|
| HTML5 | Sayfa yapısı ve semantik işaretleme |
| CSS3 | Stil, animasyon ve responsive tasarım |
| JavaScript (Vanilla) | DOM manipülasyonu ve etkileşimler |
| [Ionicons v7](https://ionic.io/ionicons) | İkon kütüphanesi |
| [ScrollReveal](https://scrollrevealjs.org/) | Kaydırma animasyonları |
| [Google Fonts](https://fonts.google.com/) | Poppins & Raleway yazı tipleri |

## 🚀 Kurulum

Herhangi bir kurulum veya bağımlılık gerektirmez!

```bash

git clone https://github.com/feridehamitoglu/food-website.git

# 2. Klasöre gir
cd food-website

# 3. index.html dosyasını tarayıcıda aç
```
